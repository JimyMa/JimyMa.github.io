单个基站下服务用户个数的统计分布规律

因为用户是随机分布的，因此在单位面积上的用户数其实是相等的，所以单个基站上有服务了多少个用户，其实可以考虑该基站的服务区域的面积。用该面积乘以用户的密度就可以求出服务的总数了。

换句话说，如果想知道单个基站下服务用户的概率质量分布函数，也只需要知道服务区域的面积就行了。因为整个小区中用户的数量已知，密度就已知了。单个基站的服务用户数的统计分布规律服从泊松分布。
